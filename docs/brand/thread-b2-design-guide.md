# THREAD B2 Design Guide

## Brand Position

THREAD

Your life. Understood. What matters next.

Thread is a memory-driven life agent.

It is not a chatbot, productivity app, or healthcare app. The product remembers, connects, notices, and helps.

## Brand Personality

Competent older sibling.

The tone is observant, proactive, calm, trustworthy, and practical. It should feel like "I've already looked into it," not "Let's optimize your life."

Avoid coach, therapist, life hacker, productivity guru, or healthcare-app language.

## Visual Territory

B2 is strongest because it feels agent-first, not dashboard-first.

The home screen order is:

1. Agent insight
2. Action queue
3. Supporting context

Avoid leading with analytics, graphs, or widgets.

## Typography

Primary UI font: GT America Mono.

Use for navigation, labels, buttons, status indicators, metadata, dates, and timelines.

Secondary display font: Noe Display.

Use for human-facing display moments such as greetings and primary statements.

In the canary app, use local licensed fonts when available and graceful CSS fallbacks otherwise.

## Color System

- Thread Black: `#0D1014`
- Night Slate: `#141A22`
- Elevated Surface: `#1B2430`
- Warm Gold: `#D8A15D`
- Seafoam: `#73C7B8`
- Periwinkle: `#8FA7FF`
- Clay: `#D98962`
- Soft Ivory: `#F5F0E8`

Warm Gold is the primary identity color. Seafoam suggests health and stability. Periwinkle suggests memory and context. Clay is for attention.

## Logo And Agent Presence

Preferred logo direction: continuity loop with one active node.

Concept: memory is continuous, the agent is active, life is ongoing.

Do not use a face, avatar, bot, or personality icon for the agent. Use the Thread Ring as the agent presence, notification indicator, memory marker, and listening state.

## Motion Language

Thread should feel like slow persistence: fade in, connect, trace, reveal.

Avoid bouncing, pulsing, gamification, or energetic motion. The UI should feel like remembering, not performing.

## Home Screen

The core home pattern:

```text
THREAD
Good morning, Alex.
I've been keeping track.
- Sleep has been below 6.5h
- Medication refill due Friday
- Insurance form still waiting
I think these may be connected.
```

Supporting sections below: Current, Timeline, Tasks, Vault.

## Photography Direction

Avoid smiling students, stock healthcare, and campus brochure imagery.

Use life as actually lived: dorm desk at night, notebook with coffee stain, laptop and class notes, apartment window, messy reality.

## Product Principle

The agent is the product.

The interface exists to support the agent's understanding of the user's life.
